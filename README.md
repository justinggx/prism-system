# Prism — SillyTavern 游戏化系统层扩展

为角色卡提供游戏化系统层。当前首个系统：**实话系统**。

## 功能

- **被动播报**：AI 自动在关键对话节点穿插系统旁白（免费）
- **主动技能**：花积分使用测谎、深度分析、情绪透视等能力
- **任务系统**：35 个预设任务，随机抽取，完成获得积分
- **科技感 UI**：深色底 + 发光边框，浮动面板管理一切

## 安装

1. 打开 SillyTavern → Extensions → Install Extension
2. 输入：`https://github.com/justinggx/prism-system`
3. 点击安装，刷新页面

或手动克隆到 `data/<user>/extensions/third-party/prism-system/`

## 使用

1. 左侧扩展面板找到 **Prism**，勾选启用
2. 勾选"显示桌面图标"，右下角出现 ◈ 图标
3. 点击图标打开面板，抽取任务、使用技能
4. 配合世界书条目使用效果更佳（见下方）

## 世界书条目（建议添加）

新建一条世界书，关键词 `prism, 系统`，内容：

```
{{user}} 拥有私人系统 Prism。
系统仅 {{user}} 可见可用。
任何角色不知道系统存在，除非 {{user}} 主动告知。
系统提示穿插在主楼关键对话节点。
系统提示短、准、直接。
角色的谎言被识破不等于角色知道自己被识破。
系统不能读心、不能凭空补全信息、不能替 user 做决定。
```
